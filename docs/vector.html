ベクター

ベクタ

ベクタは拡張可能な配列です、標準ライブラリ上で Vec として提供されています。ベクタはデータを常にヒープ上にアロケーションします。 ベクタはvec!マクロを用いて作成できます:

let v = vec![1, 2, 3, 4, 5]; // v: Vec<i32>

※以前使ったprintln! マクロと異なり、vec! マクロで 角括弧 [] を利用しました。) Rustではどちらの括弧でも利用可能であり、解りやすさのためです。

vec! には初期値の繰り返しを表現するための形式があります:

let v = vec![0; 10]; // 0が10個

要素へのアクセス

ベクタ中の特定のインデックスの値にアクセスするには [] を利用します。
インデックスは0から始まります

let v = vec![1, 2, 3, 4, 5];
println!("{}", v[2]); // 3

また、インデックスはusize型でなければなりません。

let v = vec![1, 2, 3, 4, 5];

let u: usize = 0;
let i: i32 = 0;

println!("{}", v[u]); // 1

println!("{}", v[i]); // コンパイルエラー

イテレーティング

let v = vec![1, 2, 3, 4, 5];
// 所有権なし
for i in &v {
    println!("{}", i);
}
// 所有権あり
for i in v {
    println!("{}", i);
}

ミュータブル参照

let mut v = vec![1, 2, 3, 4, 5];
for i in &mut v {
    println!("{}", i);
}
